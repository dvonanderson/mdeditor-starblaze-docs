## What is mdEditor Starblaze?

mdEditor Starblaze is a modern metadata editor application built with Ember.js. It provides a powerful, user-friendly interface for creating and managing metadata with support for various standards and formats.

## Technology Stack

- **Framework:** Ember.js 6.8 (Octane Edition)
- **Build Tool:** Vite 7.1
- **UI Framework:** Bootstrap 5.3
- **Language:** TypeScript 5.9
- **Data Layer:** Ember Data 5.6
- **Testing:** QUnit, Ember Test Helpers
- **Code Quality:** ESLint, Prettier, Stylelint, Husky
